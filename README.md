# Jarves Pro Mobile

**Jarves Pro** is a sophisticated, AI-powered mobile assistant designed to provide seamless interaction with the Gemini 1.5 Flash model. Built with Flutter, it offers a native Android experience that combines powerful core logic with a clean, professional user interface.

## Features

-   **Advanced AI Integration**: Powered by Google's Gemini 1.5 Flash model, Jarves Pro delivers intelligent, context-aware responses. It features a distinct "JARVIS" persona capable of adapting its tone based on the user's request (Formal/Professional vs. Casual/Local).
-   **Secure API Management**: Users can securely configure and store their personal Gemini API keys directly within the app settings, ensuring privacy and control.
-   **Robust Version Tracking**: The app includes a built-in version management system that tracks updates and changes via a local JSON log, ensuring transparency in development progress.
-   **Clean & Native UI**: Designed with Material 3 principles, the interface is intuitive, responsive, and free of clutter, providing a premium user experience.

## Version History & "Read More" Feature

Jarves Pro maintains a detailed log of its evolution. To view this:

1.  Tap the **Info (i)** icon in the top-right corner of the Home screen.
2.  You will see the **Current Version** and the most recent update summary.
3.  Tap the **"Read More"** button to expand the full history list. This view provides a scrollable timeline of every version release, including dates and specific changes, allowing you to track the app's journey from inception to the current stable build.

## How to Run

To build and run the application on your local machine:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/osmanghani1435/jarvis.git
    cd jarvis
    ```

2.  **Navigate to the mobile directory**:
    ```bash
    cd mobile
    ```

3.  **Install dependencies**:
    ```bash
    flutter pub get
    ```

4.  **Run the application**:
    Ensure you have an Android emulator running or a physical device connected.
    ```bash
    flutter run
    ```
